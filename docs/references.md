Markdown CLI
------------
- Requires: pandoc, lynx
- alias in ~/.bashrc:
    - `myfunc(){pandoc $1 | lynx -stdin}`
    - `alias myalias='myfunc'`
- Do not forget to enable vi-keys in lynx:
    - `o` => `[] Save to disk` => `Accept changes`
